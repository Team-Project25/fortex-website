# fortex-website

This is a responsive single-page website for the fictional company Fortex, which sells construction and renovation services. The website will showcase the company’s services, project portfolio, and provide easy ways to contact the company. The site will be built using HTML, CSS, and JavaScript, ensuring a modern, lightweight, and fully responsive design.

Features

Full-Screen Hero Section – an attractive landing section filling the viewport height
Responsive Navigation Menu – a friendly menu for easy browsing
Smooth Scrolling – animated scrolling between sections
Contact Section – includes a contact form and a map showing the company location

Project Structure

index.html – main HTML file
css/style.css – custom CSS styles
js/main.js – JavaScript for interactivity
images/ – project images
